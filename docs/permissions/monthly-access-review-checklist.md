# 월간 권한 점검 체크리스트

주기: 월 1회 · 담당: 시스템 관리자(admin) · 소요: 30~60분
결과는 날짜·점검자와 함께 보관한다.

점검일: __________  점검자: __________

## 계정
- [ ] 퇴사자 계정 비활성화(활성 계정 중 퇴사자 없음)
- [ ] 휴직자 계정 상태 확인
- [ ] 장기 미접속 관리자 확인(예: 60일+ 미로그인)
- [ ] 관리자(admin) 보유자 목록 확인(최소 인원 유지)
- [ ] 계정 공유 의심 확인(비정상 동시 접속·다지역 접속)

## 사용자 정의 권한
- [ ] 비활성 Custom Role 확인(불필요 시 정리)
- [ ] 삭제된 권한이 계정에 연결되어 있지 않은지 확인
- [ ] 중복 권한 배정 확인
- [ ] 다운로드 권한 보유자 확인(Excel/CSV/PDF)
- [ ] 전체 데이터 범위 보유자 확인(과다 여부)
- [ ] 선택(restrictive) 사용자 목록·범위 확인

## 임시/외부
- [ ] 임시 권한 종료일 도래분 회수
- [ ] 외부 협력업체 권한 유효성 확인

## 보호·감사
- [ ] 하자접수 담당자(maintenance_reporter) 메뉴 정책 유지(청소·하자만)
- [ ] 감사로그 이상 접근 확인
- [ ] 권한 없는 접근 반복 사용자 확인(정상 403 vs 비정상 구분)
