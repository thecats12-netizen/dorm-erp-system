# 분기별 보안 점검 체크리스트

주기: 분기 1회 · 담당: 보안·감사 담당 + 시스템 관리자 · 소요: 1~2시간

점검일: __________  점검자: __________

## 권한 구조
- [ ] 관리자(admin) 권한 최소화(인원·사유 재확인)
- [ ] 역할별 권한 기준(메뉴·기능·범위) 재검토
- [ ] 사용하지 않는 Custom Role 정리
- [ ] 전체 데이터 접근자 검토
- [ ] Excel/PDF/CSV 다운로드 권한 검토

## 서버 보안 (개발팀 협조)
- [ ] RLS 정책 검토(시험 공정 범위 서버 강제 정상)
- [ ] 기숙사 데이터 범위 서버 강제(RLS) 적용 여부 확인
- [ ] RPC/권한 함수 실행 권한 검토(anon 실행 권한 없음)
- [ ] Storage 접근 권한 검토
- [ ] service_role_key 클라이언트 미노출 재확인

## 감사·백업
- [ ] 감사로그 보존 상태(권한/보안 이벤트 기록 유지)
- [ ] 백업 및 복구 테스트(핵심 테이블: custom_roles, user_custom_roles, custom_role_permissions, custom_role_scopes, profiles)
- [ ] rollback SQL 최신 상태 확인

## 계정
- [ ] 로그인 실패 반복 계정 확인
- [ ] 장기 미사용 계정 확인
- [ ] 외부 협력업체 권한 유효성 확인
- [ ] 임시 계정 만료 확인
