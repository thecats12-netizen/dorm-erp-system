# 군대관리 모듈 샘플 데이터 업로드 가이드

## 개요
이 가이드는 군대관리 모듈의 자동 훈련기록 생성 기능을 테스트하기 위한 샘플 엑셀 데이터와 체크리스트를 제공합니다.

## 3. 자동 생성된 훈련기록 예상 결과

### 김준영 (예비군 1~4년차 동원)
- **자동 생성될 훈련**: 동원훈련 (28시간)
- **상태**: 일정미등록
- **비고**: 자동생성

### 이영숙 (예비군 1~4년차 동원미지정)
- **자동 생성될 훈련**: 동미참훈련 (32시간)
- **상태**: 일정미등록
- **비고**: 자동생성

### 박성철 (예비군 5~6년차)
- **자동 생성될 훈련**:
  - 기본훈련 (8시간)
  - 작계훈련 (6시간)
- **상태**: 일정미등록
- **비고**: 자동생성

### 이준호 (민방위)
- **자동 생성될 훈련**: 민방위교육 (4시간)
- **상태**: 일정미등록
- **비고**: 자동생성

### 최미영 (대상아님)
- **자동 생성될 훈련**: 없음
- **상태**: 해당없음

## 4. 업로드 체크리스트

### 4.1 인사 데이터 업로드 확인
- [ ] 인사 업로드 탭에서 `군대관리_인사_샘플.xlsx` 업로드
- [ ] "예비군/민방위 인원 관리" 탭에서 5명 모두 확인
- [ ] 각 인원의 상태, 현재구분, 동원여부 확인

### 4.2 자동 훈련기록 생성 확인

  - 김준영: 동원훈련 1건 ✓
  - 이영숙: 동미참훈련 1건 ✓

  - 이준호: 민방위교육 1건 ✓

### 4.3 훈련 시간 구분 확인
- [ ] 김준영 동원훈련: 28시간 ✓
- [ ] 이영숙 동미참훈련: 32시간 ✓
- [ ] 박성철 기본훈련: 8시간 ✓
- [ ] 박성철 작계훈련: 6시간 ✓
- [ ] 이준호 민방위교육: 4시간 ✓

## 5. 테스트 시나리오별 기대값

### 시나리오 1: 신규 인사 자동 훈련기록 생성
**입력**: 김준영 저장 (재직, 동원, 전역일 2024-03-01)
**기대값**:
- 자동생성된 훈련기록 1건 생성
- 훈련유형: "동원훈련"
- 필요시간: 28시간
- 상태: "일정미등록"

### 시나리오 2: 5~6년차 다중 훈련 생성
**입력**: 박성철 저장 (재직, 전역일 2021-08-10)
**기대값**:
- 자동생성된 훈련기록 2건 생성
- 훈련 1: 기본훈련 (8시간)
- 훈련 2: 작계훈련 (6시간)

### 시나리오 3: 상태 변경 시 관리제외 처리
**입력**: 김준영 상태를 "퇴사"로 변경 후 저장
**기대값**:
- 기존 훈련기록 상태 변경: "관리제외"
- 새로운 훈련기록 생성 안 됨
- 감사 로그에 업데이트 기록 남음

## 6. 문제 해결 가이드

### 자동 훈련기록이 생성되지 않는 경우
1. ✓ 재직상태가 자동생성 설정의 대상 상태(기본값: "재직")에 포함되는가?
2. ✓ 전역일이 올바르게 입력되었는가?
3. ✓ 군대관리 설정 탭에서 자동생성 설정이 ON 상태인가?
4. ✓ 기준연도가 올바르게 설정되었는가?

### 훈련 시간이 잘못 설정되는 경우
1. ✓ `defaultMilitaryTrainingRules`에서 각 규칙의 `requiredHours` 확인
2. ✓ 동원여부가 올바르게 인식되는가?
3. ✓ 연차 계산이 정확한가? (전역일 기준)

### 중복 생성되는 경우
1. ✓ 같은 인원+기준연도+훈련유형+차수 조합이 이미 존재하는가?
2. ✓ 브라우저 캐시 확인

## 7. 주의사항
- **필수 입력**: 이름, 재직상태, 전역일 (자동 연차 계산용)
- **대소문자 구분**: 훈련유형, 재직상태 등은 정확히 입력
- **날짜 형식**: YYYY-MM-DD (예: 2024-03-01)
- **동원여부**: "동원", "동원미지정" 또는 Y/N 인식
