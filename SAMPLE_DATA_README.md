# 군대관리 모듈 샘플 데이터 생성 및 테스트 가이드

## 🚀 빠른 시작

### 1. 샘플 엑셀 파일 생성
```bash
npm run generate-sample
# 또는
node generate-sample-excel.cjs
```

**생성 결과:**
- `public/군대관리_인사_샘플.xlsx` (5명)
- `public/군대관리_훈련기록_샘플.xlsx` (5건)

### 2. 앱 실행
```bash
npm run dev
```
http://localhost:5173 접속

### 3. 로그인
- 기본 계정: `admin` / `admin1234`

### 4. 테스트 시작
1. **인사 업로드**: 사이드바 → 군대관리 → 인사 업로드 → `군대관리_인사_샘플.xlsx` 선택
2. **결과 확인**: 군대관리 → 예비군/민방위 인원 관리 탭
3. **체크리스트**: 📄 [MILITARY_TEST_CHECKLIST.md](./MILITARY_TEST_CHECKLIST.md) 참고

## 🔍 각 기능별 검증 항목

### ✅ 1. 인사 업로드 정상 작동
```
입력: 군대관리_인사_샘플.xlsx (5명)
기대: 예비군/민방위 인원 관리 탭에 5명 표시
```

### ✅ 2. 자동 훈련기록 생성 정상 작동
```
조건:
  ✓ 자동생성 설정 ON (기본값)
  ✓ 재직상태가 자동생성 대상 상태 (기본값: "재직")
  ✓ 동일한 인원+기준연도+훈련유형+차수 중복 없음

기대:
  - 김준영: 1건 (동원훈련)
  - 이영숙: 1건 (동미참훈련)
  - 박성철: 2건 (기본훈련, 작계훈련)
  - 이준호: 1건 (민방위교육)
  - 최미영: 0건
```

### ✅ 3. 동원/동원미지정 시간 구분 정확
```
동원훈련:     28시간 ✓
동미참훈련:   32시간 ✓
(차이: 4시간)
```

### ✅ 4. 5~6년차 다중 훈련 생성 정확
```
박성철:
  - 기본훈련:  8시간
  - 작계훈련:  6시간
  - 총합:     14시간
```

### ✅ 5. 대시보드/리포트 집계 정확
```
예비군/민방위 인원 관리 탭:
  ✓ 총 인원: 5명 표시
  ✓ 각 인원의 필요훈련 정확히 계산
  ✓ 각 인원의 필요시간 정확히 표시
  ✓ 각 인원의 훈련기록수 정확 (김준영: 1, 박성철: 2 등)
```

### ✅ 6. 엑셀 업로드/다운로드 정상 작동
```
인사 정보 내보내기:
  ✓ 컬럼 구조 (이름, 계급, 군별, ..., 동원여부, ...)
  ✓ 데이터 정확성

훈련기록 내보내기:
  ✓ 컬럼 구조 (대상자, 훈련유형, 차수, 훈련예정일, 이수일, ...)
  ✓ 데이터 정확성
```

## ⚠️ 주의사항

1. **날짜 형식**: `YYYY-MM-DD` (예: 2024-03-01)
2. **재직상태 정확성**: "재직", "퇴사", "전출", "휴직" 등 정확히 입력
3. **전역일 중요**: 연차 자동 계산의 핵심 필드
4. **동원여부**: "동원" 또는 "동원미지정"으로 정확히 입력
5. **캐시 문제**: 업로드 후 브라우저 캐시 삭제 필요시 Ctrl+Shift+Delete

## 📞 문제 해결

### Q: 샘플 파일이 생성되지 않는다
```bash
# npm 스크립트로 다시 시도
npm run generate-sample

# 또는 직접 실행
node generate-sample-excel.cjs

# 오류 메시지 확인
# xlsx 패키지 설치 확인: npm install xlsx
```

### Q: 업로드 후 데이터가 보이지 않는다
1. 브라우저 개발자 도구 → 콘솔 탭 에러 확인
2. LocalStorage 확인: F12 → Application → LocalStorage
3. 권한 확인: 로그인 사용자가 "admin" 역할인지 확인

### Q: 훈련기록이 자동으로 생성되지 않는다
1. 재직상태가 "재직"인가? (기본 설정)
2. 군대관리 설정 탭에서 자동생성이 ON인가?
3. 기준연도가 올바르게 설정되었는가?
4. 브라우저 콘솔에서 오류 확인
